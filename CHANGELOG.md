# 更新日志 (Changelog)

## [v1.2.0] - 2026-03-17

### 新增功能
- ✨ 新增去水印后的 PDF 文件导出功能
- 📄 处理完成后，除了图片外，还会生成 `原文件名_已去水印.pdf` 文件

### 文件结构
```
PDF处理结果.zip
└── PDF文件名/
    ├── PDF文件名_01.png
    ├── PDF文件名_02.png
    ├── ...
    └── PDF文件名_已去水印.pdf  ← 新增
```

## [v1.1.0] - 2026-03-16

### 修复问题
- 🐛 修复上传新文件时已完成文件未清空的问题
- 🐛 修复 pdf.js worker 加载失败问题，使用本地 worker 文件

## [v1.0.0] - 2026-03-16

### 初始发布
- ✨ PDF 批量上传功能
- ✨ 智能去水印功能（支持多种水印类型）
  - 浅色半透明水印
  - 灰色半透明水印
  - 浅蓝色/灰色水印
  - 位置敏感水印检测
- ✨ PDF 转图片功能（2倍分辨率）
- ✨ 批量下载 ZIP 压缩包
- ✨ 规范化文件命名（PDF文件名_页码.png）
- ✨ 示例文件快速加载
- ✨ 实时进度显示
- ✨ 现代化 UI 设计（基于 shadcn/ui）

---

## 技术栈

- **框架**: Next.js 16 (App Router)
- **UI 组件**: shadcn/ui + Tailwind CSS
- **PDF 处理**: pdf.js + pdf-lib
- **图像处理**: Canvas API
- **文件打包**: JSZip
